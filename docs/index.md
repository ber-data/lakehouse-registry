# BER Data Registry

A metadata catalog for scientific data sources across BER-funded lakehouses at LBNL,
built with [LinkML](https://linkml.io/) and aligned with
[DCAT v3](https://www.w3.org/TR/vocab-dcat-3/) and
[DCAT-US](https://resources.data.gov/resources/dcat-us/).

## Overview

The BER Data Registry provides a standardized schema for cataloging databases and
data sources across scientific lakehouses, including the KBASE Spark lakehouse at
NERSC and Dremio-based environments. It captures ownership, access control, lifecycle
status, versioning, and technical metadata for each data source.
